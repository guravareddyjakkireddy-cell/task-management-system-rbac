# task-management-system-rbac
Full-stack Task Management System with Role-Based Access Control (RBAC), notifications, and input validations. Built using React, Node.js/Express, and MongoDB. Features include secure JWT authentication, task assignment and tracking, real-time notifications, and cloud deployment (Vercel/Render/AWS).
# Project Submission Bundle

This bundle contains ready-to-use files for the assigned tasks and final deliverables.

## Included Files
- `01_RBAC_Implementation.md` – role-based access control design and implementation notes
- `02_Advanced_Features.md` – notifications and validations implementation details
- `03_Deployment_Guide.md` – deployment steps for AWS/Vercel/other platforms
- `04_Project_Documentation.md` – complete project documentation template
- `05_Final_Presentation_Demo.md` – final presentation/demo script and slide content
- `06_Live_Project_Link_Template.txt` – placeholder file for your deployed project URL
- `07_Evaluation_Criteria_Mapping.md` – how the project addresses evaluation criteria

## How to Use
1. Open each file.
2. Replace placeholders like `[Project Name]`, `[GitHub Link]`, `[Live URL]`, and `[Your Name]`.
3. Add your real screenshots, API details, database details, and deployment information.
4. Convert the markdown files to PDF or Word if needed.
